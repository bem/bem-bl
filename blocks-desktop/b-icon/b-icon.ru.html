Абстрактная иконка

Абстрактный контейнер для иконки.

Тэг img по умолчанию выводится однопиксельной прозрачной картинкой (.gif) и пустым альтом.

Имеет опциональные атрибуты: alt, url, width, height.

Задавая атрибут url — можно подставлять b-icon новую картинку, со свомим alt, шириной и высотой.

Используя в css background: url(...); — можно использовать как спрайты, так и одиночную иконку.
