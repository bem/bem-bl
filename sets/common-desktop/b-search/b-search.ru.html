Форма поиска

{ block: 'b-search' }

﻿﻿Поисковая форма, содержащая поле ввода и кнопку.

Сам блок b-search, преобразуется в тег <form>, внутри которого лежит таблица, отвечающая за раскладку блока.
Для раскладки используется блок b-layout-table. Сам блок и его элементы cell миксируются в BEMHTML шаблоне с элементами блока b-search: [ layout-input, layout-button, layout-under ].

Рассмотрим BEMJSON блока:

block: 'b-search',
attrs: { action: '/search.xml' },
input: {
    elem: 'input',
    attrs: { value: 'Text' }
},
button: {
    elem: 'button'
},
under: [
    {
        block: 'b-link',
        url: '/',
        content: 'Advanced search'
    }
]

По умолчанию: элемент input является поисковой строкой. Элемент button — кнопкой type: 'submit'. Если вам необходимо добавить атрибуты этим элементам, достаточно в BEMJSON описать свойство attrs, и перечислить в нем все необходимые атрибуты.

Свойство under добавляет еще одну строку в раскладочную таблицу, в нее можно помещать дополнительную информацию. Например, ссылка «Расширенный поиск» под поисковым полем.
