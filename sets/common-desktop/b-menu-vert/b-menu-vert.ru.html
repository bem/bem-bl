Вертикальное меню списком

{"block":"b-menu-vert"}

Блок, строящий разметку для верстикального меню. Построен на основе блока-хелпера i-menu.

В блоке содержатся вложенные элементы двух типов:

Элементы, отвечающие за геометрию блока

  • layout
  • layout-unit

Элементы, отвечающие за функциональность блока

  • title
  • item

Элементы блока

  • item
  • item-content
    • Модификаторы элемента

* item-content

  • item-selector
  • separator
  • submenu
  • title
  • trigger

Элемент меню

{"elem":"item"}

Элемент можно содержать любой контент, чаще всего это ссылки или псевдо-ссылки. {"elem":"item-content"}

{"elem":"item-content","elemMods":{"visibility":"visible"}}

Элемент для реакции на клик

{"elem":"item-selector"}

Этот элемент появляется в мею тогда, когда требуется реакция на клик по пункту меню.
Реакция на клик может понадобиться, если при выборе пункта меню нужно что-то сделать на клиенте. Например, загрузить контент по AJAX.

Реакция есть только на клик левой кнопкой мыши. В результате клика соответствующий пункт меню (элемент item) приобретает модификатор { state: 'current'}, то есть пункт меню становится "выделенным" (текущим). Предыдущий выделенный пункт меню в то же время теряет модификатор state.
В базовой CSS-реализации блока не предоставляется код, для визуального выделения текущего пункта меню. Такой код можно реализовать на собственном уровне переопределения.

Элемент может быть представлен в DOM отдельным узлом (span) или быть смиксованным с другим элементом, чаще всего с псевдо-ссылкой.

Визуальный разделитель между пунктами меню

{"elem":"separator"}

Контейнер для пунктов вложенного меню

{"elem":"submenu"}

Используется в случае, когда необходимо представить вложенное меню часью списка меню-родителя.

Заголовок вертикального меню

{"elem":"title"}

Триггер для управления видимостью вложенного меню

{"elem":"trigger"}

Скрывает и отображает вложенное меню.

Элемент trigger может быть представлен в DOM отдельным узлом (span) или быть смиксованным с другим элементом или блоком (чаще всего - с псевдоссылкой).
